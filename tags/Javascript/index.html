Thursday 15.05. — Javascript 对象模型和原型链

重点

  • Mozilla/Konqueror/Chrome 中， 对象都有一个属性叫 __proto__，这个属性串起了整个原型链

  • 所有构造器的实例的 __proto__ 都指向这个构造器的 prototype，默认的，这个 prototype 对象也有个 constructor 属性指向构造器
